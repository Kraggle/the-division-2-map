Change Log ??st August 2019
  • The entire site has been updated to JavaScript's ES6 modules. The solitary script file was getting far too large so splitting it up just had to be done.
  • If the website won't work in a specific browser you now get a message telling you such.
  • While interacting with a layer on a touchscreen device the popups and context menus will now work correctly.
1st August 2019
  • You can now search for anything from a new menu. The search result can be sorted in a number of ways including my favourite, by distance, this creates a path through every result. Search results can be clicked which pans the map to that location. You can also search specifically by Type, Category, Content and Creator, this last one helps you to find your own layers.
  • You can now share layers and your search results, which adds a link to your clipboard.
  • There is now a context menu on all layers, this menu includes 'Search Similar', 'Share' and 'Complete' where applicable. There is also 'Settings', 'Edit' and 'Move' if you have access to do this.
  • Added a new setting to Markers so they can be cycled, this is better for Activities and the shared Resource Nodes which may be one of many taking up the same space.
  • All unsaved changes are now stored locally until you save them, this is persistent through disconnections, crashes and just forgetting to press save. Also with this there is also a cancel button, just in case you don't actually want to save your changes.
  • A bug has been fixed where switching a layer type from a layer with a popup to without would cause it to not remove the previous popup.
  • Now, when adding, removing or changing a layers type, the counts will be updated on the filter menu.
  • When duplicating a poly, as long as the type changes, it can now be left in the same place without automatically deleting.
  • You can now copy and paste layer positions, this is helpful for markers and their areas that are to be cycled.
  • Hiding Completed layers will now be persistent after reloading the page.
  • When editing a poly, if you also enable dragging, dragging will no longer work in edit mode. This stops it messing up the edit drag markers.
  • Cancel now resets a dragged layer back to it's original position.
26th June 2019
  • Changed the way the World Tiers work, now all you have to do is complete the missions to progress, this automatically adjusts the mission descriptions, icons and levels of everything on the map. Basically it works the same as the game.
  • Added a few more settings to layers for the new World Tier switching to work correctly.
  • All Side Missions will now show when on World Tier Mode, in-case you didn't do them while in Story Mode.
  • Been back over a few zones adding all of the missing materials, consumables and loot. There is still a lot more to do, but I'm getting there. Slowly!
18th June 2019
  • Updated the marker panel to display the icons in a more stylish manner.
  • Fixed the editable layers panel that wasn't displaying correctly.
  • Fixed icons not deleting when using editable layers.
  • You can now delete all editable layers.
  • Editable layers now clears when the page is reloaded, as it caused page load to increase dramatically.
  • Main and Side Missions are now grouped together in filters, respectively.
  • Added 'World Tier I' changes to missions.
5th June 2019
  • Copy and paste functionality is working better than ever.
  • Added Modes to both layers and the map for implementing World Tier changes.
  • Cookies will be used a lot less now, as we've updated to local storage.
  • Improved overall storage and loading of the database.
  • Improved a lot of functionality.
  • Fixed some existing and new bugs.
15th May 2019
  • Upon zooming out, previously hidden markers will now change to coloured dots, these the dots appear one zoom level above when they would previously be hidden. These dots are still hover-able and easily identifiable by the colour in most cases.
  • Fixed the timers not ending as they should.
  • Timers are still timed from the click rather than the new patch change of the time of day that they refresh. We still need to get those.
  • Fixed the Zone Category not showing again after hiding them all.
  • Most Collectables are now added and correctly described thanks to Engeloki.
14th May 2019
  • Improved load times by caching the bulk of the data and only downloading if it has changed.
  • Added descriptions to the settings in the tools.
  • You can now mark certain layers as complete by right clicking. With this you can...
    • show and hide any that are complete.
    • complete some temporarily, as in the caches you can collect repeatedly.
    • the timed ones have a pre-set time which matches the in-game re-spawn.
    • some have linked layers, like the Resource Nodes which will mark all associated Resources.
    • the linked layers include the areas, which will show and hide along with the markers.
  • Not all layers have been set up to mark as complete as of yet and will take time.
  • Thanks to Engeloki more markers have been added to the map and he has improved a lot of the popups.

  • A few improvements to the marker tools.
